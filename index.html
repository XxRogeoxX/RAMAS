Diccionario Git Interactiva

Aprendiendo a trabajar con ramas... ¡bajo el sol! ☀️

git init

Empezamos la aventura: Inicializa un nuevo repositorio local. ¡Prepara tu espacio de trabajo! 🏝️

git clone

Trae el paraíso a tu ordenador: Descarga una copia exacta de un repositorio remoto. 🦜

git add

Prepara las maletas: Selecciona los cambios (staging) para que estén listos para el próximo viaje (commit). 🧳

git commit

Guarda la captura de los cambios preparados en el historial
